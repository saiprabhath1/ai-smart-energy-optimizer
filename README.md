# AI-Based Smart Energy Optimizer

## Description
This project is an IoT-based system that monitors real-time energy consumption using an ESP32 and displays the data on a web dashboard. It also provides basic insights for improving energy usage.

This project was developed during my internship.

## Features
- Measures current, power, and energy
- Sends data using WiFi
- Displays data on a Flask web dashboard
- Basic AI-based suggestions

## Technologies Used
- Python (Flask)
- ESP32
- IoT (WiFi communication)

## How to Run

### Backend
Install dependencies:
pip install -r requirements.txt

Run the server:
python main_code.py

### ESP32
- Upload the code using Arduino IDE
- Update WiFi credentials
- Set the server IP address
